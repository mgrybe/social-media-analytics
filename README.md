# Dalyko "Socialinių medijų analitika" egzamino projektas (VDU, 2025)

Tai yra VDU kurso "Socialinių medijų analitika" egzamino projektas (2025), kuriame atliekama dirbtinio intelekto
mokslinių publikacijų duomenų (autorių bendradarbiavimo ir citavimo kontekste) analizė:

- Python dalyje paruošiami tekstiniai duomenys, skaičiuojami įterpiniai (embeddings), mažinamos dimensijos ir formuojami
  klasteriai.
- R dalyje konstruojami bendradarbiavimo grafai, skaičiuojami centrališkumo rodikliai ir vizualizuojami įtakingiausi
  autoriai bei bendruomenės.

Sprendimas susideda iš dviejų "Jupyter Notebook" projektų:

1. [preprocessing.ipynb](./preprocessing.ipynb) - duomenų paruošimas tinklų analizei.
2. [analysis.ipynb](./analysis.ipynb) - tinklų analizė.
